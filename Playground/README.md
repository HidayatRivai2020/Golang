# Go Playground
- Web service that runs on Google Servers
- Easiest way to test and run you Go Programs
- Write Go code, Compile, and Run
- Link : [go.dev/play](https://go.dev/play)

## Interface
- Text
    - Title
- Drop-down
    - Version
    - Example
- Button
    - Run
    - Format
    - Share
- Text-Area
    - Editor
